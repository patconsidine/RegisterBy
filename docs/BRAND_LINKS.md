## Notes

- **Breville** retired regional `/register` pages; all regions use **My Breville** (`mybreville.com`).
- **Bosch** has no single public registration form — links go to **customer services** (best available entry point).
- **Ninja AU** uses `ninjakitchen.com.au` **without** `www`.
- **Ninja UK/US** use **SharkNinja** registration (Ninja is a SharkNinja brand).
- **Dyson US** is `/registration`, not `/register`.
- **Philips** includes Philips Avent products via the same registration flows.

Source of truth in code: `RegisterBy/Services/BrandLinks.swift`
